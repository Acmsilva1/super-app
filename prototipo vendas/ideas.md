# Cake Pricing App - Design Brainstorm

## Resposta 1: Design Minimalista Moderno com Foco em Dados
**Probabilidade: 0.08**

### Design Movement
Bauhaus Digital + Data Visualization Modern

### Core Principles
- Clareza absoluta na hierarquia de informações
- Espaço negativo generoso para reduzir carga cognitiva
- Tipografia forte e contrastante para guiar leitura
- Cores neutras com acentos estratégicos

### Color Philosophy
- Fundo: Branco puro (#FFFFFF) com cinzas muito leves para áreas secundárias
- Primária: Azul profundo (#1E40AF) para ações principais
- Acentos: Laranja quente (#F97316) para valores e destaque
- Neutros: Cinzas em escala (Slate 500-900) para textos e divisórias

### Layout Paradigm
- Duas colunas assimétricas: formulário à esquerda (40%), dashboard à direita (60%)
- Cards com bordas suaves e sombra mínima
- Grid de dados com alternância de linhas claras
- Separação visual clara entre seções

### Signature Elements
- Ícones geométricos minimalistas (Lucide)
- Linhas divisórias horizontais em cinza claro
- Cards com hover elevation sutil
- Badges coloridas para status/categorias

### Interaction Philosophy
- Transições suaves (200-300ms) em todos os elementos
- Feedback visual imediato em inputs (border color change)
- Hover states com mudança de cor de fundo leve
- Animações de entrada para cards e tabelas

### Animation
- Fade-in suave para cards ao carregar
- Slide-in lateral para novas linhas de tabela
- Scale-up para botões ao hover
- Pulse sutil em valores destacados

### Typography System
- Display: Poppins Bold 32px para títulos principais
- Heading: Poppins SemiBold 20px para subtítulos
- Body: Inter Regular 14px para conteúdo
- Mono: JetBrains Mono 12px para valores monetários

## Resposta 2: Design Warm & Friendly com Tema Padaria
**Probabilidade: 0.07**

### Design Movement
Vintage Bakery + Modern Playfulness

### Core Principles
- Acessibilidade emocional através de cores quentes
- Tipografia com personalidade (fontes com serifas suaves)
- Ilustrações e elementos decorativos sutis
- Sensação de "artesanal digital"

### Color Philosophy
- Fundo: Creme suave (#FEF8F3) com texturas leves
- Primária: Marrom chocolate (#6B4423) para ações
- Acentos: Rosa coral (#F97585) e Amarelo mel (#FCD34D)
- Secundária: Verde menta (#A7F3D0) para sucesso

### Layout Paradigm
- Layout em cards flutuantes com espaçamento generoso
- Seções com cantos arredondados (16px) e sombras suaves
- Decorações sutis (linhas onduladas, pequenos ícones de bolo)
- Sidebar com navegação em abas estilizadas

### Signature Elements
- Ícones de padaria (bolo, ingredientes) em estilo flat
- Pequenas ilustrações decorativas nos cantos
- Badges com formato de fita/label
- Separadores ondulados entre seções

### Interaction Philosophy
- Animações mais lúdicas e expressivas
- Bounce effects em cliques
- Transições com easing suave (cubic-bezier)
- Micro-interações com feedback visual colorido

### Animation
- Bounce-in para cards
- Rotate-scale para ícones ao hover
- Wiggle sutil em valores de sucesso
- Slide-down para menus

### Typography System
- Display: Playfair Display Bold 36px para títulos
- Heading: Montserrat SemiBold 22px
- Body: Lato Regular 15px
- Accent: Playfair Display Italic para destaques

## Resposta 3: Design Dark Mode Profissional com Acentos Neon
**Probabilidade: 0.09**

### Design Movement
Tech-Forward Dark + Cyberpunk Subtle

### Core Principles
- Foco em redução de fadiga visual com dark mode
- Acentos neon para destaque de dados críticos
- Tipografia técnica e legível
- Contraste alto para acessibilidade

### Color Philosophy
- Fundo: Cinza muito escuro (#0F172A) com gradiente sutil
- Primária: Ciano neon (#06B6D4) para ações principais
- Acentos: Magenta (#EC4899) para alertas, Verde (#10B981) para sucesso
- Neutros: Cinzas em escala (Slate 300-700)

### Layout Paradigm
- Dashboard com grid de cards em layout 3 colunas
- Painel lateral com navegação vertical
- Cards com bordas neon suaves
- Gráficos com linhas neon em fundo escuro

### Signature Elements
- Bordas com efeito glow sutil
- Ícones com stroke neon
- Linhas divisórias em gradiente neon
- Badges com fundo escuro e texto neon

### Interaction Philosophy
- Transições rápidas (150-250ms)
- Glow effects em hover
- Feedback com cores neon
- Animações que reforçam a sensação tech

### Animation
- Glow-pulse em elementos interativos
- Slide-in com efeito de entrada tech
- Scale com glow ao hover
- Fade-in rápido para dados

### Typography System
- Display: IBM Plex Mono Bold 32px para títulos
- Heading: IBM Plex Mono SemiBold 20px
- Body: IBM Plex Sans Regular 14px
- Code: IBM Plex Mono Regular 12px para valores

---

## Decisão Final
**Escolhido: Resposta 1 - Design Minimalista Moderno com Foco em Dados**

Justificativa: Para uma aplicação de precificação, a clareza de dados é fundamental. O design minimalista garante que os usuários entendam rapidamente os cálculos e valores, enquanto os acentos em laranja criam pontos de interesse visual sem distrair. A hierarquia tipográfica forte facilita a navegação entre precificação e vendas.
